The difference between display: block, inline, and inline-block.

Display: block, inline, and inline-block all affect how content is displayed in relation to the elements around them. In short display: block creates a new line where only the element sits and pushes out an other element to another line. Automatically it takes all the available horizontal space but you can control its height and width with its height and width properties. Display: inline is the opposite, it does not push other elements to new lines or begin on a new line itself. But does not allow you to change its height and width properties. Display: inline-block is a mix of both. It is like inline in that it does not push element to a new line or begin on a new line, but does allow you to change its height and width properties.

Examples

Take aways

display: block

  • Take up all available the horizontal space of their container
  • Start on a new line
  • Push content after it to a new line
  • Have height and width properties you can change

display: inline
  • Take up only the size they need
  • Do not start on a new line
  • Does not push content after it to a new line
  • Do not have height and width properties you can change

display: inline-block
  • Don't take up all available horizontal space of thier container
  • Don't start a new line
  • Don't push content after to a new line
  • Do have a height and width properties you can change
