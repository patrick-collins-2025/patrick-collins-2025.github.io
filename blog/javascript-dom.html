Arrays and Objects

Arrays are lists of information. They have a difinitive order, starting at zero, where each item in the array has its own index. Objects also are ways to store information however the information is unordered. There are no indexes of objects, instead information is stored and retrived using key value pairs. Instead of an index infomation is stored in the object with key to be used to retrieve that information later. To access information from arrays you need to used square brackets after the arrays name, and inside the square brackets yo put the index you are accessing e.g. yourList[index]. To access information from objects you can use a fullstop after the object name, followed by the key of the key value pair you want to access e.g. yourObject.yourKey

Functions

Functions simply allow for you to avoid writting the same piece of code over and over again. Often used bits of code can be put into functions and called with functionNameHere(variable). Which will save you time, having to write the code multiple times, and will take up less memory, and be easier to change in the future.
